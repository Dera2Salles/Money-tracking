# Mitsitsy - Personal Finance Tracker

Mitsitsy is a personal finance app built with React Native and Expo. Track your expenses, manage multiple accounts, and visualize your spending habits — all 100% offline with local SQLite storage.

## Features

- Multi-account management with balance tracking
- Transaction logging with categories
- Expense charts and reports by category
- Budget planning and simulations
- Gamification system (XP, levels, streaks, badges)
- Multi-language support (French, English, Malagasy)
- Dark mode and multiple color themes
- Scheduled/recurring transactions (planifications)
- In-app notifications and tips

## Tech Stack

- **Framework**: [Expo](https://expo.dev) with [Expo Router](https://docs.expo.dev/router/introduction/) (file-based routing)
- **Language**: TypeScript
- **Styling**: NativeWind (Tailwind CSS for React Native)
- **Database**: SQLite (expo-sqlite) — 100% offline
- **State**: React Context + custom hooks
- **i18n**: react-i18next

## Get Started

1. Install dependencies

   ```bash
   pnpm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```
